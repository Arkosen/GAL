DESCRIPTION

<GAL::Reader::Template> provides << BLAH BLAH BLAH >> reading capability to GAL. It is not intended for general library use, but rather as a GAL::Reader subclass for developers of GAL::Parser subclasses. There is however no reason why it couldn't also be used as a stand alone module for other purposes.

CONSTRUCTOR

New GAL::Reader::Template objects are created by the class method new. Arguments should be passed to the constructor as a list (or reference) of key value pairs. All attributes of the Reader object can be set in the call to new. An simple example of object creation would look like this:

    $reader = GAL::Reader::Template->new(file => 'annotation_file.gff');

The constructor recognizes the following parameters which will set the appropriate attributes:

* some_attribute => $some_value

This optional attribute provides << BLAH BLAH BLAH >>

The following attributes are inhereted from GAL::Reader:

* file => feature_file.txt

This optional parameter defines what file to parse. While this parameter is optional either it, or the following fh parameter must be set before the first call to next_record.

* fh => $FH

This optional parameter provides a file handle to parse. While this parameter is optional, either it or the previous must be set before the first call to next_record.

sub next_record {
	my $self = shift;
	my $line;
	my $fh = $self->fh;
	while (my $line = <$fh>) {
	  last unless $line =~ /\s*\#/;
	}
	my @fields = split /\t/, $line;
	return wantarray ? @fields : \@fields;
}
