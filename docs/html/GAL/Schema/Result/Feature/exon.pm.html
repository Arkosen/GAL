NAME

GAL::Schema::Result::Feature::exon - A exon object for the GAL Library

VERSION

This document describes GAL::Schema::Result::Feature::exon version 0.2.0

SYNOPSIS 

    use GAL::Annotation;
    my $feat_store = GAL::Annotation->new(storage => $feat_store_args,
					  parser  => $parser_args,
					  fasta   => $fasta_args,
					 );

    $feat_store->load_files(files => $feature_file,
			    mode  => 'overwrite',
			    );

    my $features = $feat_store->schema->resultset('Feature');

    my $mrnas = $features->search({type => 'mRNA'});
    while (my $mrna = $mrnas->next) {
      my $exons = $mrna->exons;
      while (my $exon = $exons->next) {
	my $id    = $exon->feature_id;
	my $start = $exon->start;
	my $end   = $exon->end;
      }
    }

DESCRIPTION

<GAL::Schema::Result::Feature::exon> provides a <GAL::Schema::Result::Feature> subclass for exon specific behavior.

METHODS

There are currenlty no exon specific methods implimented for <GAL::Schema::Result::Feature::exon>. See <GAL::Schema::Result::Feature> for inhereted methods.
