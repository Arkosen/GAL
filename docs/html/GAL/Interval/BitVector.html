NAME

GAL::Interval::BitVector - Provide functions for intervals on sequence

VERSION

This document describes GAL::Interval::BitVector version 0.2.0

SYNOPSIS 

    use GAL::Interval::BitVector;

DESCRIPTION 

    GAL::List::Nuimeric provides a collection of methods for intervals
    on sequence.  It uses Bit::Vector under the hood to provide the
    functionality.
